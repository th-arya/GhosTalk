# Requirements

## v1 Requirements

### Authentication
- [ ] **AUTH-01**: Users receive auto-assigned ghost identities upon entry without signing up.

### Realtime Messaging
- [ ] **CHAT-01**: Users can join temporary chat rooms.
- [ ] **CHAT-02**: Users can send and receive messages in real-time.

### Data Retention
- [ ] **DATA-01**: Messages auto-delete after 4 hours.

## Deferred (v2)

- Custom usernames
- Private rooms

## Out of Scope

- User accounts with passwords or emails — to ensure zero-friction, fully anonymous usage constraint.
- Complex media sharing (images/video) — to stick within the 4-hour hackathon constraint and zero cost limit.
